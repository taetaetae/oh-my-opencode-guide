# 변경 로그

## v3.2.1 (현재)

이 문서는 Oh-My-OpenCode v3.2.1을 기준으로 작성되었습니다.

### 주요 기능

- 11개 전문 에이전트 시스템
- 34개 라이프사이클 훅
- 20+ 도구 (LSP, AST-Grep, 위임 등)
- 3개 내장 MCP 서버
- 4개 내장 스킬

### 문서화된 내용

- 에이전트 시스템 전체 문서화
- 스킬 시스템 및 커스텀 스킬 작성법
- 슬래시 명령어 및 커스텀 명령어 작성법
- 훅 시스템 및 Rules 파일 작성법
- MCP 통합 가이드
- 설정 옵션 전체 문서화
- 8개 워크플로우 예제
